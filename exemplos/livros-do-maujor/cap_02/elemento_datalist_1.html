Elemento Datalist Html

Criado para ser usado em conjunto com o elemento input. Destina-se a fornecer uma lista de sugestões para preenchimento do campo input e permite que o usuário selecione uma das opções de uma lista.

Resultado do Exemplo abaixo



Exemplo DATALIST Html

      <form action="#">
         <label for="fru">Fruta preferida: </label>
         
         O FOR DO LABLE LIGA-SE AO ImPUT PELO ID
         
         <input type="text" list="frutas" id="fru">
         
         O LIST DO INPUT LIGA-SE AO DATALIST PELO ID
            <datalist id="frutas">
               <option value="Abacate"></option>
               <option value="Abacaxi"></option>
               <option value="Banana"></option>
               <option value="mamão"></option>
               <option value="manga"></option>
               <option value="Pera"></option>
         </datalist>
      </form>
