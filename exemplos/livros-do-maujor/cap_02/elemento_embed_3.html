A Tag Embed Html

Destina-se a servir de container para incorporar na página aplicações ou conteúdos externos interativos, tipicamente não escritos em HTML.

O elemento embed admite os seguintes atributos: atributos globais, src, type, width, height.

O atributo src é obrigatório e define o URL (endereço) em que se encontra o conteúdo externo a incorporar na página.

O atributo type define o tipo, os atributos width e height definem as dimensões largura e altura da área.
