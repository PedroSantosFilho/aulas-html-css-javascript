Elemento dialog Html

Destina-se a criar uma caixa de diálogo, uma caixa de inspeção ou uma janela modal de interação com o usuário, e admite o atributo global open.

Exemplo dialog Html

   <dialog>
      <h3>Janela modal</h3>
      <p>Conteúdo da janela modal</p>
      <button id="fechar">X</button>
   </dialog>

   <button id="abrir">Abrir janela modal</button>

Exemplo JavaScript

   
   <script>
      let janela = document.querySelector('dialog');
      document.querySelector('#abrir').onclick = 
      function() { 
         janela.show(); // abrir a janela modal
      };
      document.querySelector('#fechar').onclick = 
      function() {
         janela.close(); // fechar a janela modal
      };
   </script>
