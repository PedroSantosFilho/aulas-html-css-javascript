Inserindo Vídeo 01 Sem CONTROLS Html

Vídeo sem Controles

Como não foram definidos controles para o vídeo é possível que na plataforma e/ou navegador que se esteja visualizando o exemplo não haja controles para interagir com o vídeo. Se este for o caso clique com o botão direito do mouse sobre a área do vídeo e escolha a opção no menu de contexto

Exemplo Html Vídeo

O elemento HTML <video> </video> é usado para mostrar um vídeo em uma página da web.

O Atributo SRC

O atributo src destina-se a indicar o caminho para o arquivo de vídeo a incorporar na página.

Caminho para o arquivo

Este é o caminho "/codigos/cap_03/video.ogv" atribuido ao SRC onde se encontra o arquivo do vídeo que será exibido. 

   <video src="/codigos/cap_03/video.ogv"></video>
