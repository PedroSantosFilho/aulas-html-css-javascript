Vídeo Com SOURCE Html

O Elemento Html VÍDEO

O elemento HTML <video> é usado para mostrar um vídeo em uma página da web.

O Atributo SRC

O atributo src destina-se a indicar o caminho para o arquivo de vídeo a incorporar na página.

Caminho para o arquivo

Este é o caminho "/codigos/cap_03/video.ogv" atribuido ao SRC onde se encontra o arquivo do vídeo que será exibido.

O Atributo CONTROLS

O atributo controls destina-se a fazer com que o navegador renderize os controles nativos para o usuário interagir com o vídeo.

O Atributo AUTOPLAY

Para iniciar um vídeo com Reprodução automática, assim que a página for renderizada, use o atributo autoplay, com ou sem o atributo controls.

O Atributo LOOP

Os atributos autoplay e loop destinam-se respectivamente a iniciar o som automaticamente, tão logo a página seja carregada e a repetir o som indefinidamente.

O Elemento Html VÍDEO e seus AtributoS

   <video 
   src="/codigos/cap_03/video.ogv" controls autoplay loop >
   </video>

O elemento Html SOURCE

A tag <source> é usada para especificar vários recursos de mídia para elementos de mídia, como <video> , <audio> e <picture>

A tag <source> permite que você especifique arquivos alternativos de vídeo/áudio/imagem que o navegador pode escolher, com base no suporte do navegador ou na largura da janela de visualização. O navegador escolherá o primeiro <source> que suportar.

Atributos da Tag SOURCE

Note que a tag <source> possui três atributos:

‣ type
vai indicar o media type da imagem que usamos (veja mais informações sobre media types no capítulo 10)
‣ srcset
vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido
‣ media
indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo srcset.

Vídeo com o elemento SOURCE

      <video  controls autoplay>
         <source src="/codigos/cap_03/video.ogv" type="video/ogv">
         <source src="/codigos/cap_03/video.mp4" type="video/mp4">
         <source src="/codigos/cap_03/video.webm" type="video/webm">
         <source src="/codigos/cap_03/video.m4v" type="video/m4v">
      </video>
