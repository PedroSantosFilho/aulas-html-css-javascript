Inserindo Vídeo 04 Com MUTED Html

O Elemento Html VÍDEO

O elemento HTML <video> é usado para mostrar um vídeo em uma página da web.

O Atributo SRC

O atributo src destina-se a indicar o caminho para o arquivo de vídeo a incorporar na página.

Caminho para o arquivo

Este é o caminho "/codigos/cap_03/video.ogv" atribuido ao SRC onde se encontra o arquivo do vídeo que será exibido.

O Atributo CONTROLS

O atributo controls destina-se a fazer com que o navegador renderize os controles nativos para o usuário interagir com o vídeo.

O Atributo AUTOPLAY

Para iniciar um vídeo com Reprodução automática, assim que a página for renderizada, use o atributo autoplay, com ou sem o atributo controls.

O Atributo LOOP

Os atributos autoplay e loop destinam-se respectivamente a iniciar o som automaticamente, tão logo a página seja carregada e a repetir o som indefinidamente.

O Elemento Html VÍDEO e seus AtributoS

   <video 
   src="/codigos/cap_03/video.ogv" controls autoplay loop >
   </video>
