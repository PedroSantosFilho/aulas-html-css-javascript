Elemento Html Áudio

O elemento ÁUDIO destina-se a incorporar um som ou um STREAm de áudio

O elemento audio admite como elemento-filho o elemento source cuja finalidade é substituir o atributo src e permitir formatos alternativos para o som.

Exemplo 1 o elemento audio

Sintax: elemento AUDIO, O atributo SRC e entre aspas o endereço onde se encontra o audio.

      <audio src="/codigos/cap_03/som.mp3">
      </audio>

Exemplo 2 o elemento audio e o elemento source

O elemento source admite os seguintes atributos globais: src, type, e media.

O atributo ( src ) aponta para o endereço do arquivo de som, ( type ) Informa ao navegador o tipo de mídia a ser incorporado, ( media ) Informa ao navegador o tipo de mídia. 

   <audio>
      <source src="/codigos/cap_03/som.ogg" 
         type="audio/ogg" 
         codecs="vorbis" 
         media="screen">
      <source src="/codigos/cap_03/som.mp3" 
         type="audio/mpeg" 
         codecs="mp3" 
         media="screen">
         <!--Código (x)Html para inserção 
         de som.mp3 para flash-->
   </audio>

Exemplo 3 o elemento audio e o atributo controls

O elemento audio admite os seguintes atributos globais: src, controls, crossorigin, preload, autoplay, mediagroup, loop e muted.

O atributo controls destina-se a fazer com que o navegador renderize os controles nativos para o usuário interagir com som. 

   <audio controls>
      <source src="/codigos/cap_03/som.ogg" 
         type="audio/ogg" 
         codecs="vorbis" 
         media="screen">
      <source src="/codigos/cap_03/som.mp3" 
         type="audio/mpeg" 
         codecs="mp3" 
         media="screen">
      <source src="/codigos/cap_03/som.wav" 
         type="audio/wave">
      <!--Código (x)Html para inserção 
      de som.mp3 para flash-->
      <p>
         Se seu navegador não suporta o 
         elemento audio Html:5. 
         Faça <a href="som.mp3">download 
         do som.mp3</a>
      </p>

Mas, cuidado. Marcação (X) HTML inserida no elemento audio como mostrado anteriormente não cumpre finalidade de oferecer alternativa textual para o som, pois ela é renderizada somente em navegadores que não suportam o elemento audio.

Exemplo 4 o elemento ÁUDIO

E os atributos: controls, autoplay e loop

Os atributos autoplay e loop destinam-se respectivamente a iniciar o som automaticamente, tão logo a página seja carregada e a repetir o som indefinidamente. 

   <audio controls autoplay loop>
      <source src="/codigos/cap_03/som.ogg" 
         type="audio/ogg" 
         codecs="vorbis" 
         media="screen">
      <source src="/codigos/cap_03/som.mp3" 
         type="audio/mpeg" 
         codecs="mp3" 
         media="screen">
      <!--Código (x)Html para inserção 
      de som.mp3 para flash-->
      <p>
         Se seu navegador não suporta 
         o elemento audio Html:5. 
         Faça <a href="som.mp3">download 
         do som.mp3</a>
      </p>
   </audio>
